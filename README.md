# Ânima Educação - Landing Page

Esta é uma landing page moderna e responsiva desenvolvida para a **Ânima Educação**. O projeto foca em apresentar a instituição como uma líder em educação digital, utilizando tecnologias atuais para uma experiência de usuário fluida.

## 🚀 Tecnologias Utilizadas

- **HTML5**: Estrutura semântica do conteúdo.
- **CSS3 (Customizado)**: Estilização detalhada com foco em design moderno e animações.
- **Tailwind CSS**: Framework utilitário para agilidade no layout responsivo.
- **JavaScript (Vanilla)**: Interatividade, scroll suave e animações baseadas em observação de interseção.
- **Framer Motion**: Biblioteca para animações avançadas (via CDN).
- **Google Fonts**: Tipografia profissional (Poppins e Montserrat).

## 🛠️ Funcionalidades

- **Design Responsivo**: Adaptado para dispositivos móveis, tablets e desktops.
- **Scroll Suave**: Navegação fluida entre as seções da página.
- **Animações ao Scroll**: Elementos que surgem suavemente conforme o usuário navega.
- **Seções Informativas**:
  - Hero com call-to-action.
  - Estatísticas de impacto.
  - Sobre a instituição (Missão e Visão).
  - Áreas de estudo com cards interativos.
  - Diferenciais competitivos.
  - Depoimentos de alunos.
  - Rodapé completo com links e contatos.
